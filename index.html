How to Use DevSpace

  1. Write Code: Use the three editor panels on the left to write your HTML structure, CSS styling, and JavaScript logic.
  2. Run: If "Auto-Run" is on, changes appear automatically. Otherwise, click the Run button.
  3. Debug: Open the Console (Bug icon) to check for errors or logs. Open the Terminal (Square icon) to run system commands like `clear` or `reset`.
  4. Export: Happy with your creation? Click the download icon to save your work as an HTML file to your device.
